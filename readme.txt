This plugin will make sure the /ads.txt route of your website will be up to date with the latest ads.txt file from Publisher Collective.

This plugin will fetch content from the Publisher Collective ads.txt file once a day here: https://kumo.network-n.com/adstxt/partner.txt

You can find the Publisher Collective terms of service here: https://www.network-n.com/terms-and-conditions/

You can find the Publisher Collective privacy policy here: https://www.network-n.com/privacy-policy/
